# Starcat Pro

Starcat Pro 是 Starcat 的公开支持与发布说明仓库。

Starcat 是一款原生 macOS 应用，用于管理 GitHub Stars、阅读仓库上下文，并通过 AI 辅助整理有价值的项目。

## 这个仓库用来做什么

- 通过 GitHub Issues 收集公开 bug 反馈和功能建议。
- 维护 Starcat 唯一可信的公开更新日志。
- 把面向用户的支持材料和私有应用源码分离。

## 这个仓库不包含什么

- 不包含应用源码。
- 不包含后端源码。
- 不包含构建脚本或私有配置。

如果要反馈 bug、体验问题或功能建议，请创建 GitHub Issue。反馈 bug 时，建议附上 Starcat 版本、macOS 版本、复现步骤，以及可见的错误信息。
