# Starcat Pro

Starcat Pro is the public support and release-notes repository for Starcat.

Starcat is a native macOS app for managing GitHub Stars, reading repository context, and organizing useful projects with AI-assisted workflows.

## What This Repository Is For

- Collect public bug reports and feature requests through GitHub Issues.
- Publish Starcat release notes in one canonical changelog.
- Keep user-facing support material separate from the private app source tree.

## What This Repository Does Not Contain

- No application source code.
- No backend source code.
- No build scripts or private configuration.

Please open a GitHub Issue for bugs, usability problems, and product suggestions. When reporting a bug, include your Starcat version, macOS version, steps to reproduce, and any visible error message.
